## Install dependencies

```bash
cd frontend
npm install

cd backend
npm install
```

## Steps to build App

1. build static frontend

```bash
cd frontend
npm run build
```

2. copy folder ./dist from frontend then paste it in backend

3. build backend server

```bash
cd backend
npm run build
```

4. App will host in <a href="localhost:3000" target="_blank">localhost:3000/</a>
